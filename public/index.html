Crossplay Dictionary Search

Use this website to search for words in the NYT Crossplay dictionary. By default it only finds exact matches (similar to in-app functionality), but it has a 'rack search' mode that interprets '?' as a blank tile and tries all permutations of letters in your rack to find all available words. There is a limit of two blanks as that's the maximum number of blanks in Crossplay. Each blank significantly slows the search as the number of possibilities grows quickly, so please be patient with 'two blank' searches. This app runs purely in your browser, once the dictionary is loaded you don't have to be online to access this tool.
